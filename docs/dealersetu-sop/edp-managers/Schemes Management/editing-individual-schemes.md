# Editing an Individual Scheme

This guide explains how an **EDP (Electronic Data Processor)** can edit an existing scheme in the DealerSetu system.

## Overview

If there are any changes in the scheme for a specific vehicle, you can update it by editing the individual scheme instead of performing a bulk import.

## Step-by-Step Process

### Step 1: Click the Edit Icon

![Updating individual schemes Step 1](/img/dealersetu/edp/editing_scheme/step1.png)

Locate the scheme you want to update and click the **Edit** icon under the **Actions** column.
