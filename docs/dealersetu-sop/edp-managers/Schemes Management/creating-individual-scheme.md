# Creating a New Scheme

This guide explains how an **EDP (Electronic Data Processor)** can create a new scheme in the DealerSetu system.

## Overview

When a new scheme needs to be added for a specific vehicle, you can create it directly instead of using the bulk import option.

## Step-by-Step Process

### Step 1: Click on Create

![Creating new schemes Step 1](/img/dealersetu/edp/creating_scheme/step1.png)

Ensure you are on the **Schemes** tab before proceeding.

---

### Step 2: Fill All Mandatory Fields

![Creating new schemes Step 2](/img/dealersetu/edp/creating_scheme/step2.png)

Enter all required scheme details accurately.

---

### Step 3: Click on Save

![Creating new schemes Step 3](/img/dealersetu/edp/creating_scheme/step3.png)

Click **Save** to create the new scheme.
