# Printing Gate Pass Manually

This guide explains how an **EDP (Electronic Data Processor)** can **manually print a gate pass** when a vehicle has been processed manually in the **DealerSetu** system.

## Overview

Use this process to **manually generate and print a gate pass** in **DealerSetu**.

## Step-by-Step Process

### Step 1: Open Gate Pass Menu

![Printing gate pass manually Step 1](/img/dealersetu/edp/manual-gatepass/step1.png)

Navigate to the **Gate Pass** section from the dashboard.

---

### Step 2: Click on Create

![Printing gate pass manually Step 2](/img/dealersetu/edp/manual-gatepass/step2.png)

Click on **Create** to generate a new manual gate pass.

---

### Step 3: Fill in the Details

![Printing gate pass manually Step 3](/img/dealersetu/edp/manual-gatepass/step3.png)

Enter all required vehicle and customer details.  
Fields marked with **\*** are **mandatory**â€”the gate pass cannot be generated unless these fields are filled correctly.

---

### Step 4: Click on Print

![Printing gate pass manually Step 4](/img/dealersetu/edp/manual-gatepass/step4.png)

Click **Print** to generate a **ready-to-print gate pass**.
