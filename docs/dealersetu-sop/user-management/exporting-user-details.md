# Exporting Users as Excel

This guide explains how to **export user details as an Excel file** in **DealerSetu**.

## Overview

You can **download the complete list of users** in **DealerSetu** with a **single click** using the export feature.

## Step-by-Step Process

### Step 1: Click on Export as Excel

![Exporting User Details Step 1](/img/dealersetu/superAdmin/exporting_user_details/step1.png)

Click on the **Export** button to download the **user list in Excel format**.
