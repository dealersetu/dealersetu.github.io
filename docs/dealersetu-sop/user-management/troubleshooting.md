# Troubleshooting

Common issues and solutions for DealerSetu.

## Common Issues faced by Super Admins

### Issue 1: Location not avilalable in the dropdown while createing or updating user

**Problem:** When we create a new user or update the existing user there is a Dropdown for selecting location, but it is not showing the desired location. 

**Solution:** This happens because the desired location is not added in DealerSetu yet. Ask EDP to create a new Yard of that location, once created you will see that location in the dropdown.


## FAQ

### How to delete a user incase someone switched roles or organization?

You cannot delete the user, however you can Edit it and updates the details by replaceing all the fields including employee ID

### Can we create two or more users with same name and roles?

Yes you can create two or more users with same name and roles, but the employee ID must be Unique.
